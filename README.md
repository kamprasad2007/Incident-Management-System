# Incident Management System

This project has 3 main services. Front-end, Backend and database. All these run on individual docker container and docker compose combines all these 3.

## Build Services
    docker-compose build

## Run Services
    docker-compose up

## Access Website
    http://localhost:3000

## Port details
    Front-End   3000
    Backend     4000
    Database    5432
